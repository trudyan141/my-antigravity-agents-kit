# Antigravity Personal Agent Kit

A powerful, customizable **Agentic OS** structure designed to turn any AI assistant into a professional software engineer. This kit allows you to maintain consistent rules, specialized skills, and complex workflows across any project.

## ðŸ“¦ Installation

Install the `.agent` folder directly into your project using `npx`:

```bash
# Navigate to your project
cd your-project

# Install .agent folder
npx @trudyan141/antigravity-kit init
```

This command will:

- Copy the `.agent` folder to your current project.
- Automatically update your `.gitignore` to keep internal brain files out of your project's repository.
